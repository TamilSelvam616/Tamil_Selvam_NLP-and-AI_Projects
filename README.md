# Practical 3 - Word Frequency and Vocabulary Analysis

# Aim:

To perform word count and frequency distribution on a given text corpus to understand common words and vocabulary size.

# Procedure

Install and import the NLTK library.

Download the required punkt dataset.

Load a sample text corpus.

Tokenize the text into words.

Use FreqDist to calculate word frequencies.

Find the most common words.

Count the total vocabulary size.

Display frequency table.

Analyze frequently appearing words.

Save the output for further NLP processing.

Program executed successfully.
