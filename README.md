6.Parts Of Speech(POS)

# Aim: 

To identify the grammatical roles (noun, verb, adjective, etc.) of words in a sentence using POS tagging.

# Procedure:

Import required NLTK libraries.

Download punkt and averaged_perceptron_tagger.

Load a sample sentence.

Tokenize the sentence.

Apply pos_tag() to assign tags.

Display word-tag pairs.

Identify common POS tags such as NN, VB, JJ.

Save results for further NLP tasks.

Repeat with different sentences.
