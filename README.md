# Practical 8 – Text Classification using Machine Learning

# Aim:
To build a simple text classifier (e.g., spam detection ) using machine learning.

# Procedure:

Import required libraries.

Load a small labeled dataset.

Preprocess text by converting to lowercase.

Convert text to numerical features using Bag of Words.

Split dataset into training and testing sets.

Train Multinomial Naive Bayes classifier.

Predict class labels for testing data.

Evaluate model using accuracy score.

Display predicted and actual labels.

Save the trained model.

# Result:
Program executed successfully.
