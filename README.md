# Practical 1 - Introduction to Python for Al and NLP

# Aim:

To write basic Python programs using data types, control structures, and functions to prepare for Al and NLP tasks.

# Procedure:

Install Python 3.x from the official website.

Install Jupyter Notebook or VS Code for writing Python programs.

Create variables using different data types such as integers, floats, strings, lists, tuples, and dictionaries.

Apply control structures using if, else, and elif.

Use looping statements like for and while.

Execute the program and verify the output.

Debug errors if any.
