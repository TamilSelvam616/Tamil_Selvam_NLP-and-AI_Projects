# Practical 10 – Rule-Based Chatbot
# Aim:
To design a simple rule-based chatbot using pattern matching to answer user queries.

# Procedure:

Import required libraries re and random.

Create a dictionary of user patterns and responses.

Preprocess user input (convert to lowercase).

Match input using regular expressions.

Display corresponding response.

Provide default reply if no match found.

Loop chatbot for continuous interaction.

Test chatbot with various inputs.

Save the script for future use.
