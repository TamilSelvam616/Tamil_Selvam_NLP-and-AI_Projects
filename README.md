# Practical 2 - Text Preprocessing Techniques

# Aim:

To preprocess text data using tokenization, stop-word removal, stemming, and lemmatization for NLP tasks.

# Procedure:

Install NLTK using pip command.

Download required NLTK datasets such as punkt, stopwords, and wordnet.

Load a sample text corpus.

Perform tokenization to split the text into words.

Remove stopwords to eliminate frequently occurring words.

Apply stemming to reduce words to their root form.

Apply lemmatization to convert words into dictionary form.

Display intermediate and final results.

Program executed successfully.
