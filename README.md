# Practical 15  - Text Summarizer
# Aim:

To generate a concise summary of a large text using NLP techniques.

# Procedure:

Import nltk and heapq.

Load or input a text document.

Tokenize text into sentences and words.

Compute word frequencies, ignoring stopwords.

Score each sentence based on important words.

Select top sentences to form the summary.

Display summarized text.
