# Practical 5 – Text Vectorization Techniques (BoW and TF-IDF)
# Aim:

To implement Bag of Words (BoW) and TF-IDF vectorization techniques to convert text into numerical feature vectors for NLP modeling.

# Procedure:

Import CountVectorizer and TfidfVectorizer.

Load a sample text corpus.

Preprocess text (lowercasing and tokenization).

Apply BoW using CountVectorizer.

Convert BoW to feature matrix.

Apply TF-IDF using TfidfVectorizer.

Convert TF-IDF to feature matrix.

Display feature matrices.

Compare importance of words.

Save vectors for further NLP tasks.

Program executed successfully.
