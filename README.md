# Practical 7 – Named Entity Recognition (NER)

# Aim:

To identify named entities such as Person, Location, Organization and Date in a given text using Natural Language Processing (NLP) techniques.

# Procedure:

Install and import NLTK library.

Download required datasets: punkt, averaged_perceptron_tagger, maxent_ne_chunker, words.

Load a sample input text.

Perform sentence segmentation.

Apply word tokenization.

Apply POS tagging.

Perform Named Entity Recognition using ne_chunk().

Identify named entity labels such as PERSON, ORGANIZATION, GPE, DATE.

Display extracted named entities.

Save results for NLP applications

Program executed successfully.
